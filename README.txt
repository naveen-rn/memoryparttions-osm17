This repository contains the source for the OpenSHMEM-17 workshop paper.
The main gist of the paper is about proposing multiple symmetric heap
feature called Symemtric Memory Partitions in OpenSHMEM.

Directory structure:
bmark - micro-benchmarks and test cases used in the paper
data  - raw data from the performance benchmarks
gplot - gnu plot scripts to create graphs
graph - graphs created from gnu plot
image - png images used in the paper
src   - main source of the paper
        content - all the main sections and sub-sections of the paper
        example - example codes/algorithms used inside the paper
        utils   - utility sources like the common usage of terms and
                  terminologies in the paper
